# LeetCode 刷题项目

## 📖 项目简介

这是一个用于系统化学习 LeetCode 算法题目的项目。通过**启发式教学**的方式，逐步建立算法思维，而不是简单地记忆答案。

## 🎯 学习理念

### 启发式学习，而非直接给答案

本项目采用**苏格拉底式提问法**，通过以下步骤引导学习：

1. **问题分析** - 深入理解题目要求和约束条件
2. **思路探索** - 从暴力解法开始，逐步优化
3. **复杂度分析** - 分析时间复杂度和空间复杂度
4. **代码实现** - 逐步实现，遇到困难时给予提示
5. **代码优化** - 讨论改进方案和最佳实践
6. **总结归纳** - 提炼解题模式和通用技巧

## 💻 支持的编程语言

- **主语言**: TypeScript
- **计划支持**: Java, Go, Python

## 🚀 学习流程

### 开始新题目

1. **告诉 AI 你要刷哪道题**
```

我想刷 LeetCode 的第 X 题：[题目名称]

```

2. **AI 会引导你思考**
- 首先理解题目
- 探索可能的解法
- 分析复杂度
- 逐步实现代码

3. **完成后总结**
- AI 会帮你把整个思考过程和代码总结到 `src/X-题目名称.md`
- 记录关键点和易错点

### 复习已完成的题目

- 定期回顾 `src/` 目录下的题解
- 尝试用其他语言重新实现
- 思考是否有更优的解法

## 🎓 学习原则

1. **不要急于看答案** - 先自己思考，哪怕只能想到暴力解法
2. **理解比记忆重要** - 搞清楚为什么这样做，而不是记住怎么做
3. **多种解法对比** - 理解不同解法的权衡
4. **定期复习** - 算法需要反复练习才能内化
5. **举一反三** - 总结题型和解题模式
